🛒 E-commerce SQL Project

📊 Description
   This project is based on an E-commerce database where I performed data analysis using SQL. It includes multiple tables and real-world queries to understand customer behavior and sales.

🛠️ Tools Used
  * MySQL Workbench
  * SQL
  
📁 Tables Created
   * Customers
   * Products
   * Orders
   * Order_Items
  
🔍 Key Operations Performed
   * Created multiple tables with relationships
   * Inserted realistic sample data
   * Used SQL JOINs to combine data
   * Performed aggregation using SUM, COUNT
   * Applied GROUP BY and ORDER BY clauses

📈 Analysis Performed
  * Total sales calculation
  * Most sold product
  * City-wise order analysis

🚀 Outcome
   This project helped me understand how to work with multiple tables, perform joins, and analyze real-world business data using SQL.
